# InvestSim Pro

Simulador de investimentos com aportes mensais e efeito bola de neve.

## Funcionalidades

- Carteiras pré-definidas: Brasil, EUA e híbridas
- Simulação de crescimento de patrimônio no longo prazo
- Estimativa de renda futura
- Ajuste de aportes mensais, taxa anual e tempo
- Dados históricos de preços via Yahoo Finance

## Tecnologias

- Python 3.11+
- [Streamlit](https://streamlit.io/)
- [yfinance](https://pypi.org/project/yfinance/)
- Pandas
